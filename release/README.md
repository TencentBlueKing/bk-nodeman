## 此目录为版本日志集合

### 版本号说明
{major_version}.{minor_version}.{build_version}

### 版本日志文件名
文件名仅记录到`minor_version`，`{build_version}`在版本日志内容中详细记录
```
V{major_version}.{minor_version}.0_{date}.md
```

### 版本日志内容

```markdown
### 【V{major_version}.{minor_version}.{build_version}】{date}
----
【新增】AAA
【修复】BBB
【优化】CCC
【更新】DDD
```
